# Rock Paper Scissors

A command-line Rock Paper Scissors game built with Node.js and TypeScript. Created as a playground for experimenting with [Claude Code](https://claude.ai/claude-code).

## Requirements

- Node.js 24+
- pnpm

## Running the game

```sh
pnpm start
```

Type `rock`, `paper`, or `scissors` to play a round. Type `quit` to exit.

## Running the tests

```sh
pnpm test
```
